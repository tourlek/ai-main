กองทุนสำรองเลี้ยงชีพ ไทยพาณิชย์ มาสเตอร์ฟันด์ — เริ่มสมัคร ก.ค. 2567

**แผนลงทุนที่เลือก:** แผนที่ 5 — ตราสารหนี้ 60% / SET Index 30% / SCBNDQ(A) 10%
**อัตราเงินสะสมลูกจ้าง:** 5%
**อัตราเงินสมทบนายจ้าง:** ตรงกับฝั่งลูกจ้าง (2%, 5%, 10%)

เงื่อนไขรับเงินสมทบคืน (ตามอายุสมาชิก):
- ≤1 ปี: 0%
- >1–3 ปี: 20%
- >3–4 ปี: 50%
- >4–5 ปี: 75%
- >5 ปี: 100%

อายุสมาชิกนับจาก ก.ค. 2567 → ครบ 5 ปี ก.ค. 2572 (รับ 100%)

**Why:** บันทึกเพื่อใช้วางแผนสภาพคล่อง กรณีพิจารณาลาออกจากงานหรือถอนเงินกองทุน
**How to apply:** ก่อนคำนวณเงินที่จะได้รับเมื่อออกจากงาน ให้ดึง % อายุสมาชิก ณ วันนั้นมาคำนวณก่อน
